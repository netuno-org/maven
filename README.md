# maven
Java Maven projects integrated with the Netuno Platform.

This repository hosts code for the Maven tutorial in this [link](https://doc.netuno.org/docs/academy/server/java-core/maven-setup/).

It also contains code for the tutorials on Java Core [Service](https://doc.netuno.org/docs/academy/server/java-core/web-service), [Resource](https://doc.netuno.org/docs/academy/server/java-core/resource) and [Event](https://doc.netuno.org/docs/academy/server/java-core/event).
